Option B Source Icon + Text in Info Area

Small icon paired with the source name, placed in the card's info area. Muted grey so it doesn't compete with price or address. Icons are simple and recognizable — a house for Realtor, a list for Craigslist, etc.

Option C Multi-Source Composite Pills

Each source appears as a small colored pill with a dot indicator. Multi-source listings show all sources side by side, making Dwelligence's aggregation visible and a feature — not hidden. Single-source listings show just one pill.

Option D Favicon-Style Tiny Icons

Minimal 14x14 colored icons at the bottom of each card — almost invisible until you look for them. Hover reveals a tooltip with the source name. Multi-source listings show a row of icons. The most low-profile option — no text, no visual noise.
